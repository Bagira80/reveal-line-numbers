AJAX Framework Overview

AJAX framework was introduced in Drupal 7. To Dynamically update data. Allows events to trigger callbacks to the server. A PHP macro language, returns json object with javascript instructions. Json can contain markup or array of commands.


first take a step back and review the AJAX Framework.

  • Used to dynamically update data.
  • Events trigger callback functions.
  • Server returns json object of commands.

More info: http://bit.ly/DrupalAJAX

  • Render arrays use the '#ajax' property.
  • Form submits use the class 'use-ajax-submit'.
  • Links use the class 'use-ajax'.

What are AJAX Commands?

A breakdown

  • A bridge between server and client.
  • Are returned by an AJAX callback.
  • Wrappers for jQuery manipulation methods.

Drupal 7 Core Commands

  • 17 AJAX callback commands.
  • Commands defined in "includes/ajax.inc".
  • Javascript defined in "misc/ajax.js".
  • Full list: http://bit.ly/D7ajaxCMD

Drupal 7 Command Examples

ajax_command_replace($selector, $html, $settings = NULL); 


                ajax_command_replace('#my-div', '
...
');

ajax_command_css($selector, $argument); 


                ajax_command_css('#my-div', 'background-color:#00f;');

ajax_command_invoke($selector, $method, $arguments = array()); 


                ajax_command_invoke('#my-div', 'fadeIn', array(2000));

Drupal 8 Core Commands

  • 22 AJAX callback commands.
  • Commands located in "core/lib/Drupal/Core/Ajax/".
  • Implement CommandInterface.php.
  • Javascript defined in "misc/ajax.js".
  • Full list: http://bit.ly/D8ajaxCMD

Drupal 8 Command Examples

RestripeCommand($selector); 


              new RestripeCommand('#my-table');

ReplaceTitleCommand($title); 


              new ReplaceTitleCommand('My New Title');

RedirectCommand($url); 


              new RedirectCommand(Drupal::url('node/12', array(), array('absolute' => TRUE)));
